# odin-recipes
This is my first HTMl project from The Odin Project Foundations course.This project involves creating a simple recipe website using only HTML. The site will feature a main homepage that links to several individual recipe pages. Each recipe page will include a recipe title, an image, a brief description, a list of ingredients, and detailed step-by-step instructions.

Through this project, I will demonstrate foundational HTML skills, including building a proper HTML document structure, using headings and paragraphs, adding links and images, creating ordered and unordered lists, and organizing files within a project. I will also gain experience using Git and GitHub for version control, project management, and tracking changes throughout development.
